Expected Results:

The error message: “Invalid credentials” should be displayed, because the entered username ("mariana") does not exactly match the registered username ("Mariana") due to the expected case-sensitivity.

Actual Results:

The user successfully logs in and is redirected to the system's Dashboard.

Steps to Reproduce:

·  Ensure a user account exists with a username containing at least one capitalized letter, e.g., Mariana.

· Navigate to the Login Page: https://opensource-demo.orangehrmlive.com.

·  Enter the Username: mariana (all lowercase).

· Enter the Password: m123.

· Click the Login button.

 

 Description 

 

 

The case-sensitivity test for the username field failed. The system allowed the user to successfully log in using a lowercase username ("mariana") when the correct, registered username was mixed-case or capitalized ("Mariana"). This behavior violates the expected security requirement for username case-sensitivity.

· Priority: [ HIGH ]

· Severity: [ MEDIUM ]
